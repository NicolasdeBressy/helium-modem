1 - Créer du bon contenu
      - 400 mots par pages (minimum)
      - Garder un contenu à jour
      - Ne pas répéter toujours les mêmes choses (ne pas spammer le robot Google)
      - Ne pas faire une page d'intro avant le Site
      - Pas de lien "cliquez ici" "en savoir plus"
      - Penser à être local (préciser ville, quartier, région), le robot Google préviligie les choses locales. Mettre .fr si votre site est destiné qu'à la France.

2 - HTML "seo friendly"
      - voir HTML Helium
      - Open Graph : https://ogp.me/
      - Tester open graph: https://cards-dev.twitter.com/validator 

3 - Site rapide (performant)
      - Le robot test le temps de chargement
      - Il faut que le JS soit lu en dernier
      - Il faut faire apparaitre le plus vite possible une couleur de fond
      - Priviligier les images depuis le CSS (et non pas la balise IMG)

Tester son SEO : plugin "Lighthouse"
